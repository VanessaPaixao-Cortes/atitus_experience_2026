## 🚀 Protocolo Phoenix – Cidade 0

Sistema de validação de códigos desenvolvido para a experiência imersiva interdisciplinar **ATITUS EXPERIENCE**, no formato escape room educacional.

A aplicação faz parte da narrativa **Protocolo Phoenix – Cidade 0**, criada pela **ATITUS Educação**, e simula a reativação do núcleo central de uma cidade futurista em colapso após um cenário de guerra.

Durante a experiência, os participantes percorrem diferentes estações e coletam códigos ao resolver desafios relacionados às áreas do conhecimento da instituição. Ao final, esses códigos são inseridos no sistema. Quando a combinação correta é validada, o sistema é reativado e um vídeo final é exibido, simbolizando a estabilização da cidade.

### 💡 Funcionalidades

* Inserção de 4 códigos provenientes das estações
* Validação automática dos códigos
* Feedback visual de erro ou sucesso
* Tela imersiva de **“Acesso Liberado”**
* Reprodução automática do vídeo final
* Efeito sonoro de validação

### 🎯 Objetivo

Proporcionar uma finalização imersiva para a experiência **Protocolo Phoenix – Cidade 0**, reforçando a proposta pedagógica da **ATITUS**, que integra diferentes áreas do conhecimento na resolução de problemas complexos e reais.

### 🛠️ Tecnologias

* HTML
* CSS
* JavaScript
